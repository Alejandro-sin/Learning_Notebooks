    Etiqueta Video

    Permite cargar videos renderizables. Posee un atributo src para decirle donde se encuentra. Para ponerle botones y poder manipular mi video, debo usar otros atributos. Controls, es un atributo de una palabra, no existe un valro para este. El atributo preload permite que el video se renderiza cuando la pagina se carga en el navegador, es una forma de entregar los recurosos. No quiere decir que el video se repreduzca solo. La reproducción automática es una mala práctica y se penaliza en los navegadores.

    Para ajusta el intervalo de reporducción añadimos una sintáxxis que nos permite ajustar esto: "./video/fuchi.mp4#t=10,60" Donde # Referencia a la función del intervalor t de tiempo (inicial, final).

    Cuando quiero que me cargue diferentes formato debo usar dentro de la etiqueta video una llamada source, para darle diferentes fuentes a los diferentes tipos de video y así rendereizar alguno de los formatos
